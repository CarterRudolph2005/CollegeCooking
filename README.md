
<h1>College Cooking - Website Prototype</h1>

<h2>General Description</h2>
Project is built using HTML, CSS, and Javascript to create a enjoyable experience finding easy recipies that have flexible ingredients lists.

<h2>Languages Used</h2>

- <b>HTML</b> 
- <b>CSS</b>
- <b>JavaScript</b>

<h2>Technical Skills</h2>
<p>- Used HTML to define the structure of the webpage.</p>
<p>- Used CSS to format the styles of each component.</p>
<p>- Used JavaScript for the dynamic menue and recipie display cards.</p>
